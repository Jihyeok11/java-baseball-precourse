- [x] 랜덤 숫자 중복 확인 기능

- [x] 랜덤 숫자 세 개 생성 기능

- [x] 숫자 입력 받는 기능

  - 같은 숫자 없을 경우 : `낫싱`

- [x] 결과 출력 기능
